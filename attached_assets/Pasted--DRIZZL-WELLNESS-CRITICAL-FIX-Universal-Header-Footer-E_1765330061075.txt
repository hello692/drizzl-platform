## STEP 6: MAKE IT MAINTAINABLE

**Best Practice:** Create reusable header/footer components

### Option A: Include Files (if using PHP or similar)
```php
<?php include 'includes/header.php'; ?>
<!-- Page content here -->
<?php include 'includes/footer.php'; ?>
```

### Option B: JavaScript Template (if using plain HTML)
```javascript
// Load header and footer dynamically
fetch('components/header.html')
  .then(response => response.text())
  .then(html => document.getElementById('header').innerHTML = html);

fetch('components/footer.html')
  .then(response => response.text())
  .then(html => document.getElementById('footer').innerHTML = html);
```

### Option C: Copy-Paste (if no build system)
If you can't use includes:
- Keep one master copy of header in a comment
