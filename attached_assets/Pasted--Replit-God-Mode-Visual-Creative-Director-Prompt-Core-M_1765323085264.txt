## Design Principles

### 1. Create Flow Through Motion
- Use scroll animations to reveal sections progressively
- Stagger list items as they enter viewport
- Add smooth parallax to backgrounds
- Create momentum with directional animations (left→right, bottom→top)

### 2. Build Depth Without Color Changes
- Layer sections with varying shadows
- Use elevation to create hierarchy
- Add subtle background patterns
- Create visual separation through spacing

### 3. Dynamic Visual Rhythm
- Alternate section layouts (full-width, contained, offset)
- Vary animation timings and directions
- Create breathing room with generous spacing
- Use asymmetry for visual interest

### 4. Decorative Enhancements
- Add abstract shapes in backgrounds
- Include flowing SVG dividers between sections
- Use animated gradient backgrounds
- Add subtle particle effects or floating elements

## Animation Examples

### Entrance Animations
- Fade up: opacity 0→1 + translateY(50px→0)
- Fade right: opacity 0→1 + translateX(-50px→0)
- Scale in: opacity 0→1 + scale(0.8→1)
- Blur in: filter blur(10px→0)

### Continuous Animations
- Floating: gentle up/down movement
- Rotating: slow continuous spin
- Pulsing: subtle scale breathing effect
- Gradient shift: animated background position

### Hover Effects
- Lift: translateY(0→-10px) + shadow increase
- Glow: box-shadow expansion
- Tilt: subtle 3D rotate effect
- Scale: transform scale(1→1.05)
