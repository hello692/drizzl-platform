3) JS to make mouse wheel scroll horizontally:

const scroller = document.querySelector('.stories-scroller');

if (scroller) {
  scroller.addEventListener(
    'wheel',
    (e) => {
      if (Math.abs(e.deltaY) > Math.abs(e.deltaX)) {
        e.preventDefault();
        scroller.scrollLeft += e.deltaY;
      }
    },
    { passive: false }
  );
}

This should produce a smooth, horizontal, mouse-scrollable image strip with no visible card boxes, preserving the full composition of each image.
