# Drizzl Didone Typeface

A high-contrast Didone-style display typeface inspired by luxury fashion typography.

## Design Characteristics
- High stroke contrast (thin hairlines vs thick stems)
- Vertical stress axis
- Unbracketed hairline serifs
- Elongated ascenders
- Low x-height ratio
- Elegant, refined letterforms

## Usage
Import the SVG sprite or use individual letter components.
